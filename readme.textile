h2. requirements

this project requires "buildr":http://buildr.apache.org/:
<pre>
rvm use jruby
gem install buildr
</pre>

it is tested on OS X Snow Leopard only

h2. buildr

buildr can run the tests for you:
<pre>
buildr clean test
</pre>

you can use buildr to integrate into eclipse:
<pre>
buildr eclipse
artifacts:sources
</pre>
